# Bash Scripts

This folder contains basic Bash automation scripts.

## create_folders.sh
Creates 5 folders, adds a text file inside each folder, and lists the folder structure.

## check_file.sh
Checks whether a file named `test.txt` exists.

## disk_check.sh
Checks disk usage and prints a warning if usage is over 80%.

## Skills Practiced

- Bash scripting
- for loops
- if/else conditions
- chmod permissions
- df, awk, sed
- basic DevOps monitoring
