# DevSecOps Learning

This repository contains my practice scripts for Linux, Python automation, Git, and DevSecOps learning.

## Scripts

### count_lines.py
Reads a text file and counts the total number of lines.

### api_test.py
Calls the GitHub API using the requests library and prints the response.

### os_test.py
Uses the os module to print the current working directory.

### subprocess_test.py
Runs Linux commands from Python using the subprocess module.

### scan_py_files.py
Scans folders and lists all Python files using os.walk().

## Skills Practiced

- Linux terminal basics
- Python file handling
- API requests
- os module
- subprocess module
- folder scanning automation
- Git and GitHub basics
